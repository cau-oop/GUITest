대신 오른쪽에 탭창에서 한가지 이상을 골라 텍스트창에 각 조건 입력 후 그때그때 확인 버튼을 누르면 체크박스가 자동으로 체크됩니다. 확인 버튼을 그때그때 반드시 눌러야 합니다.

사업자로 로그인 시 뜨는 창에서 패키지 추가 버튼을 누르면 패키지를 등록할 수 있는데 왼쪽에 있는 버튼 (패키지 이름 버튼 ~ 최대인원 버튼)을 하나씩 눌러 뜨는 텍스트 창에 해당 정보를 입력해준 후

이때 등록 버튼을 누르면 내부적으로 GUITest폴더에 있는 packagelist.txt에 입력한 패키지 정보가 저장됩니다. 

여행자와 가이드로 로그인 시 뜨는 창에서 채팅 버튼은 아직 CLI로 구현했지만 GUI로 추가하지 못한 채팅기능을 위한 버튼입니다. CLI로 구현한 채팅 프로그램을 따로 첨부했습니다.

채팅 프로그램 실행 방법은 먼저  압축파일을 풀고(채팅 메신저 기능(추후 GUI로 구현 예정).zip),
chatting_server.exe를 서버역할을 할 컴퓨터에서 실행해놓습니다.
실행 방법 - cmd 창에서 chatting_server.exe를 다운 받은 위치로 가서  ./chatting_server.exe 9290 를 입력.  * 9290은 열 임의의 포트번호.

그리고 클라이언트 역할을 할 컴퓨터에서 chatting_clnt.exe를 실행합니다.
실행방법 - cmd 창에서 chatting_server.exe를 다운 받은 위치로 가서 ./chatting_clnt.exe 서버IPv4주소 9290 아이디 사용자type 을 입력.
이때 아이디는 예를 들어 kkt752 와 같은 것이고,
사용자 type은 가이드 또는 일반회원과 같은 것입니다.

이렇게 서버 프로그램을 한 컴퓨터에서 실행시키고 클라이언트 역할을 할 컴퓨터 2대 이상에서 chatting_clnt.exe를 실행시키면 데모 동영상에서 처럼 두개 이상의 클라이언트 컴퓨터에서 채팅이 가능합니다.
